# MiniStats

> Lightweight desktop overlay for real-time system monitoring

## ✨ Features

- CPU & GPU temperature display
- Clean and minimal UI
- Customizable layout
- System tray integration
- Ultra lightweight

## 🚀 Usage

1. Start the application  
2. Right-click → **Settings**  
3. Configure your overlay  

## 🖥 Requirements

- Windows 10+
- .NET 8 Runtime
